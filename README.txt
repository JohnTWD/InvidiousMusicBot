This is a Discord Bot that does the following:
	- Tracks Invidious playlists for changes, for example, if a video is added/removed
	- Plays Invidious audio to Discord voicechats for listening


API Documentation
https://docs.invidious.io/api/

Setup instructions:
	- `python3 -m venv bot-env`
	- `pip install -r requirements.txt`
